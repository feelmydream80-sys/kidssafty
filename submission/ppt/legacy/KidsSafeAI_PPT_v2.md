# 슬라이드 9: AI 기술 도입 (실제 구현 시작)
---
### 왜 Random Forest(랜덤 포레스트)를 선택했을까요?
**💡 랜덤 포레스트란?**
- 수백 개의 **결정 트리(Decision Tree)**를 만들고, 각 트리의 예측을 투표로 모아 최종 결과를 내는 AI 방식
- 비유: **100명의 전문가에게 각각 의견을 묻고, 다수결로 결정**하는 것과 같음

**✅ 우리 시스템에 적합한 이유 (3가지)**
1. **이해하기 쉬움**: "왜 이곳이 위험한가?"를 **특징 중요도**로 설명 가능 (예: 횡단보도 밀집 35% 기여)
2. **작은 데이터에 강함**: 사고 데이터가 9개뿐이지만, 트리 100개가 각기 다른 관점에서 학습해 **안정적 예측**
3. **다양한 데이터 처리**: 숫자(거리, 건수)와 범주(신호등 유무)를 동시에 잘 처리

### Random Forest로 얻은 효과
| 효과 | 설명 | 수치 |
|------|------|------|
| **높은 정확도** | 기존 사고지역 9곳을 99.5% 일치하게 예측 | R² = 0.995 |
| **원인 분석** | 어떤 요인이 위험을 높이는지 파악 → "횡단보도 밀도 30% 기여" | 특징 중요도 제공 |
| **신규 위험지역 발견** | 기존 통계에 없던 새로운 위험 지역 6곳 추가 발굴 | 6개 클러스터 |
| **빠른 응답** | 새 데이터 입력 시 즉시 예측 (0.1초 미만) | 실시간 적합 |

### 실제 구현 결과
- **모델**: RandomForestRegressor (n_estimators=100)
- **학습 데이터**: Positive(위험) 9개소 주변, Negative(안전) 2,050개 격자점
- **저장 및 시각화**: `data/predicted_risk_zones.json`, `combined.html` 주황색 원 표시
