# 어린이 사고 다발 구역 경고 앱 개발 계획

## 1. 프로젝트 개요

- **프로젝트명**: Kids Safety Alert
- **유형**: PWA (Progressive Web App)
- **핵심 기능**: 사고 다발 구역을 지도에 표시하고, 사용자가 해당 지역 진입 시 GPS 기반 경고

## 2. 데이터 개요

- **소스 파일**: `12_24_child.csv`
- **레코드 수**: 47개
- **지역 분포**: 서울, 경기, 부산, 대구, 경남 등
- **데이터 필드**:
  - 사고 지점 ID, 코드
  - 지점명, 상세주소
  - 사고 통계 (사상자, 사망자, 부상자 등)
  - **중심 좌표**: 경도(lon), 위도(lat)
  - **폴리곤 영역**: GeoJSON Polygon 형태

## 3. 기술 스택

| 구분 | 기술 |
|------|------|
| 플랫폼 | PWA (크로스플랫폼 웹앱) |
| 프레임워크 | Vanilla JS |
| 지도 | Leaflet.js + OpenStreetMap (무료, 회원가입 불필요) |
| 위치 추적 | Geolocation API |
| 알림 | Web Notification API |
| 빌드 | 정적 HTML (파일 서버) |

## 3.1 사용자 정책

| 항목 | 내용 |
|------|------|
| 회원가입 | 불필요 (Anonymous 사용) |
| 데이터 저장 | LocalStorage (설정, 위치 등) |
| 계정 연동 | 없음 |

## 4. 주요 기능

### 4.1 지도 표시
- Leaflet.js 기반 인터랙티브 지도
- 사고 지점 마커 표시 (아이콘: 경고삼각형)
- 마커 클릭 시 상세 정보 모달

### 4.2 반경 영역 설정
- 사용자 정의 반경 선택 (기본값: 300m)
- slider 또는 입력 필드로 반경 조정
- 선택된 반경의 원형 오버레이 시각화

### 4.3 GPS 기반 경고
- 실시간 위치 추적 (Background Geolocation)
- 보호 구역 진입 감지 시:
  - 경고음 재생
  - 진동 알림
  - 화면 알림 (Push 또는 인앱)
  - 근처 사고 지점 정보 표시

### 4.4 PWA 기능
- 오프라인 캐싱 (Service Worker)
- 홈 화면 설치 지원
- 백그라운드 위치 추적

## 7. 개발 단계

1. **Phase 1**: 프로젝트 세팅 + Leaflet 지도 연동
2. **Phase 2**: CSV 데이터 파싱 및 마커 표시
3. **Phase 3**: 반경 선택 UI + 원형 오버레이
4. **Phase 4**: GPS 추적 + 경고 로직 구현
5. **Phase 5**: PWA 설정 + Service Worker
6. **Phase 6**: 테스트 및 배포

## 8. 참고 사항

- 한국 지도 데이터 정확도: OpenStreetMap + Leaflet 활용
- 실제 배포 시: HTTPS 필수 (Geolocation API 보안 요건)
- 백그라운드 추적: 브라우저 제한으로 완전히 Background 동작은 어려울 수 있음

## 9. 개발 모드

### 9.1 모드 종류

| 모드 | 설명 |
|------|------|
| **일반 모드** | 실제 GPS 좌표 사용 |
| **테스트 모드** | 마우스로 지도上の 위치 드래그 이동 가능 |

### 9.2 테스트 모드 기능

- 지도 마우스 드래그로 사용자 위치 설정
- 현재 위치 마커를 드래그하여 이동
- 화면 우하단에 테스트 모드 토글 버튼 표시

## 10. 경고 처리

- 브라우저 Notification API 사용
- 진동 (Vibration API - 모바일만)
- 화면 내 Alert Banner 표시
- 근처 사고 지점 정보 Popup

## 11. 추가 기능

### 11.1 기본 위치 설정

- 초기 지도 중심: `[37.54, 127.01]` (서울 동작구/용산구 주변)
- 사고 다발 구역이 가장 밀집한 지역

### 11.2 사고 지역 목록 패널

- 지도 우하단에 표시
- 사용자 위치 기준 가까운 순 정렬 (최대 10개)
- 실시간 거리 표시 (m/km 단위)
- 클릭 시 해당 지역으로 지도 이동

### 11.3 점선 연결선

- 사용자 위치 → 최근접 5개 사고 지역까지 점선 연결
- 선상 거리 레이블 표시
