# 동작구 교통사고 데이터 수집 가이드

## 🎯 목표
현재 `data/child_safety_zones.csv`는 시간 정보가 없음.
→ 3년 치(2023~2025) 동작구 교통사고 데이터 수집하여 시간대별/월별 분석 가능하게 함.

## 📂 데이터 소스별 수집 방법

### 1. TAAS (교통사고분석시스템) - 권장 ⭐⭐⭐⭐⭐
**URL**: https://taas.koroad.or.kr/sta/acs/exs/typical.do?menuId=WEB_KMP_OVT_UAS_PDS

**수집 절차**:
1. 회원가입 (개인/연구자 가능)
2. 로그인 → "교통사고 상세통계" 메뉴
3. 검색 조건 설정:
   - 지역: 서울특별시 > 동작구
   - 기간: 2023-01-01 ~ 2025-12-31
   - 사고유형: 보행자 사고 (어린이/노인 포함)
4. "목록보기" 클릭 → "엑셀 다운로드" 버튼
5. 파일명: `TAAS_동작구_보행사고_2023-2025.xlsx`

**포함 컬럼**:
- 사고번호, 발생일시, 발생시간, 발생요일
- 위도, 경도 (좌표)
- 사망자수, 중상자수, 경상자수
- 도로형태, 신호상태, 기상상태 등

### 2. 서울 열린데이터광장 - 구별 통계 ⭐⭐⭐
**URL**: https://data.seoul.go.kr/dataList/10057/S/2/datasetView.do

**수집 절차**:
1. "서울시 교통사고 현황(구별) 통계" 접속
2. "CSV파일 내려받기" (연도별로 2023, 2024, 2025 각각)
3. 파일명: `seoul_accident_gu_2023.csv`, `seoul_accident_gu_2024.csv`, ...

**주의**: DB 형태로 제공되어 직접 다운로드가 안 될 수 있음.
→ "Open API 신청" 메뉴에서 API 키 발급 후 데이터 수신.

### 3. 공공데이터포털 - TAAS 연동 ⭐⭐⭐⭐
**URL**: https://data.go.kr/data/15075666/fileData.do

**수집 절차**:
1. 공공데이터포털 회원가입
2. "한국도로교통공단_교통사고 상세통계" 데이터 신청
3. 승인 후 다운로드 (XLS 형식, 318회 다운로드됨)
4. 시트별로 필터링: "서울특별시" > "동작구"

## 📊 필요한 컬럼 (체크리스트)
- [ ] 발생일시 (YYYY-MM-DD HH:MM:SS)
- [ ] 발생시간 (HH:MM)
- [ ] 발생요일 (월~일)
- [ ] 시도시군구명 (동작구)
- [ ] 위도, 경도
- [ ] 사고건수
- [ ] 사망자수, 중상자수, 경상자수
- [ ] 도로형태 (교차로, 직선도로 등)
- [ ] 신호상태 (정지, 횡단중 등)

## 🔧 수집 후 처리 (Python 코드 예시)
```python
import pandas as pd

# TAAS 데이터 로드
df = pd.read_excel('TAAS_동작구_보행사고_2023-2025.xlsx')

# 시간 정보 추출
df['발생일시'] = pd.to_datetime(df['발생일시'])
df['년'] = df['발생일시'].dt.year
df['월'] = df['발생일시'].dt.month
df['시간대'] = df['발생일시'].dt.hour

# 동작구만 필터링
dongjak = df[df['시도시군구명'].str.contains('동작구')]

# 저장
dongjak.to_csv('data/dongjak_accidents_with_time.csv', encoding='utf-8', index=False)
```
